## Étapes:
1. **Déclenchement serveur laravel**
    - Navigation vers la source du projet Laravel
    ```
    php artisan serve
    ```

1. **Déclenchement serveur React**
    - Navigation vers la source du projet Laravel
    ```
    npm run dev
    ```
    - Ouverture du port 3000
    ```
    http://localhost:3000/login
    ```
